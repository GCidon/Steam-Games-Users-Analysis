Dataset description

We started working with multiple datasets from kaggle but we finally decided to work with a dataset from steamspy, which offered us the information we wanted for our study. This dataset contains information about more than 25000 videogames and data about their sales, playtime, tags, developers... Using this data, we managed to compare and get information about thousands of applications in just a few seconds.
